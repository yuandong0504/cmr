This directory contains Requests for Comments (RFCs) for the
Capability Message Runtime (CMR).

Each RFC documents a single, concrete design decision or experiment
that advances the system toward semantic closure.

RFCs are:
- narrowly scoped
- explicitly motivated
- technically verifiable
- historically permanent

They are not proposals for discussion,
but records of decisions that have been validated in code.
